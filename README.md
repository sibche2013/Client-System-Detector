## English

A highly precise, modern, and zero-dependency **client environment inspector tool** built using **Vanilla JavaScript**. It parses user device parameters, network connectivity flags, live browser properties, and utilizes modern API fallbacks to distinguish hidden environmental states like Windows 11.

### 🔗 Live Preview
You can test the live version of this project here:  
👉 **[Launch Live Demo](https://demo.aminarjmand.com/Client-System-Detector/)**

### ✨ Key Features
- **Windows 11 High-Accuracy Fix:** Integrates the modern `navigator.userAgentData` API to circumvent the native User-Agent legacy locking bug that incorrectly classifies Windows 11 as Windows 10.
- **Hardware Metrics Tracker:** Displays monitor screen resolutions instantly inside clear responsive blocks.
- **Live Battery Monitoring Subsystem:** Fetches real-time terminal battery levels and active AC power connections using the Battery Status API.
- **Network State Observer:** Hooks live async structural flags tracking internet drops or active connection returns immediately.
- **Premium Frosted UI:** Restructured layout with a responsive Glassmorphism container featuring dedicated metric indicators.
